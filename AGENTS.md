# Hermes Agent

**Recipe Runner** ships a Hermes skill. Your agent should run the install playbook in **`docs/HERMES-AGENT.md`** (clone → venv → `scripts/install-hermes-skill.sh` → verify).

Quick command after clone:

```bash
cd recipe-runner && bash scripts/install-hermes-skill.sh
```

Then ask Hermes to load skill **`recipe-runner`** and add a recipe (URL, paste, or photo).
